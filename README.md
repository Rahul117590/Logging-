## 1. Git & GitHub Workspace Setup

Follow these structured commands to set up your repository and initialize it within your local development environment:

### Step 1: Create a Repository on GitHub
1. Navigate to your GitHub account and click on **New**.
2. Name your repository and click **Create repository** (Keep it blank without initializing README/License if you are pushing local code).

### Step 2: Clone and Initialize in VS Code
Open your terminal and execute the following sequential workflow:

```bash
# Clone the remote repository to your local workstation
git clone https://github.com/Rahul117590/Logging-.git

# Move inside the cloned project directory
cd LOGGING

# Initialize Git locally (if creating a brand new repository from scratch)
git init

# Launch the current workspace inside Visual Studio Code
code .
